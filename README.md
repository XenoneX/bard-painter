Рисовалка для плана помещения
============

Для работы с рисовалкой у Вас должен быть на странице div или любой другой элемент для заполнения с фиксированным размером. Для работы необходимо в <head> загрузить файл main.js. Файл tools.js является вспомогателльным. В нем присутствуют дополнительные функции, такие как получение результатов рисования и прочее.
